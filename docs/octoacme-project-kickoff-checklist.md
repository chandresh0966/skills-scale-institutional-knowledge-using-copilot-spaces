# OctoAcme — Project Kickoff Checklist

## Purpose
Provide a reusable, role-assigned checklist to ensure all critical activities are completed before and during a project kickoff. Use this alongside the [Project Initiation Guide](./octoacme-project-initiation.md).

## When to use
At the start of any new project or significant workstream, after the Project One-pager is approved and before sprint planning begins.

---

## Pre-Kickoff Preparation

| # | Action | Owner | Done? |
|---|--------|-------|-------|
| 1 | Complete and share Project One-pager (problem, goal, success metrics) | Product Manager | [ ] |
| 2 | Identify and confirm all core team members and role assignments | Project Manager | [ ] |
| 3 | Validate technical feasibility and flag initial technical risks | Technical Lead | [ ] |
| 4 | Prepare initial UX/design direction or reference designs (if applicable) | Design Lead | [ ] |
| 5 | Identify stakeholder groups and assign a Stakeholder Champion | Project Manager | [ ] |
| 6 | Confirm environment and infrastructure readiness | Operations Specialist | [ ] |
| 7 | Set up project board, repo, and documentation structure | Project Manager | [ ] |
| 8 | Distribute kickoff agenda and pre-read materials to attendees | Project Manager | [ ] |

---

## Post-Kickoff Actions

| # | Action | Owner | Due |
|---|--------|-------|-----|
| 1 | Publish meeting notes and decisions | Project Manager | Within 24 hours |
| 2 | Finalize and baseline project plan and milestones | Project Manager | Within 3 days |
| 3 | Create initial risk register | Project Manager / Technical Lead | Within 3 days |
| 4 | Set up recurring ceremonies (standups, sprint planning, retrospectives) | Project Manager | Within 3 days |
| 5 | Begin backlog grooming and sprint 0 planning | Product Manager | Within 1 week |
| 6 | Confirm design tooling and share initial design assets | Design Lead | Within 1 week |
| 7 | Set up monitoring, alerting, and deployment pipeline skeleton | Operations Specialist | Within 1 week |
| 8 | Confirm stakeholder communication cadence and calendar invites | Stakeholder Champion / Project Manager | Within 3 days |
| 9 | Complete project board setup and assign initial tasks | Project Manager + Developers | Within 1 week |

---

## Decision Gate — Ready to Execute?

Move to execution (sprint 1) when:
- [ ] Project plan and milestones are agreed and shared.
- [ ] All team roles are filled and individuals are confirmed.
- [ ] Initial backlog is prioritized and sprint 1 items are estimated.
- [ ] Risk register is created and initial mitigations are documented.
- [ ] Development and deployment environments are available.
- [ ] Stakeholder communication plan is confirmed.
